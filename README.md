# Quaratined Gamers Discord Bot

## Command Prefix
Use '>' before text to interact with the bot.

Example:\
`>Yo`\
`>Hi`\
`>Bye`

## Memes
The Bot can display memes on different topics using simple command.\
`>meme (Topic name)`
